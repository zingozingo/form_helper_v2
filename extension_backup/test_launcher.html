Chrome Extension

To test the Form Helper extension with the test forms:

  1. Load the extension in Chrome Developer mode
  2. Open the test forms page
  3. Observe how the extension detects and scores each form
  4. Adjust the threshold in content.js to experiment with detection sensitivity
